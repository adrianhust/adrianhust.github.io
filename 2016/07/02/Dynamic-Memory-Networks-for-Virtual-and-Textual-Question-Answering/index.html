Dynamic Memory Networks for Virtual and Textual Question Answering

  • QA, attention, memory network

地址： http://arxiv.org/pdf/1603.01417v1.pdf

最近QA很火，跟chatbot 一样。
相比传统DMN，作者提出了DMN+；对于输入数据，利用bi gru，来fuse 处理后的输入数据；输入数据 会经过encoding 得到 sentence的 embedding，送给bigru；对于视觉数据，也就是图片，通过cnn 提取局部特征，然后通用是利用bigru 做fusion，理由是单个局部描述特征不具有全局信息，因此性能不够好，比较有限。最后用memory network 关联问题和facts，基于attention 机制用question 挑选 facts, 来回答问题
